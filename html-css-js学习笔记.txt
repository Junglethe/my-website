第一部分：html

HTML（HyperText Markup Language，超文本标记语言）它没有逻辑控制，所以它不是编程语言
HTML通过标签来标记不同的内容，标签有双标签和单标签（自闭合标签）
双标签：开始标签+内容+结束标签   eg:<h1>这是一个双标签</h1>   开始和结束只有一个'/'区别
单标签：只有一个标签，没有内容    eg:<br>
标签还能添加属性（Attributes），提供额外信息，写在<>（开始标签或单标签）里面，用空格分隔

框架：

<!DOCTYPE HTML>    //严格的说，这一条不是标签，而是一条声明，声明这是一个HTML5文档
<html>                       //整个文档的起点
	<head>    
	
	</head>
	<body>
	
	</body>
</html>

<html>常见属性：语言属性：作用：定义文档语言      eg:<html lang="en"></html>    en:英文  zh-CN:简体中文  fr:法语

<head>标签没有常用属性，它主要是个容器，包含其它标签，包含文档的元信息，这些信息不会直接显示在网页上，而是被浏览器、搜索引擎或其他工具使用。
一些常用或基本的内容：
<head>
	<title>标题</title>           //该标签的内容会出现在浏览器顶部（一般是在顶部）的网页标签里
	<meta charset="utf-8">   //meat标签是一个比较重要的标签，它的charset属性用来定义网页字符编码，这里使用utf-8编码，以下几行都是它的一些常见用法
	<meta name="keywords" content="HTML, 学习, 教程">  //当前页的关键字，搜索引擎输入关键字时搜索结果更有可能出现这个网站:（感觉说的不清楚
	<meta name="author" content="猫娘">                            //当前页的作者名
	<meta name="description" content="这是一个HTML教程的网页">//当前页的内容描述
	<meta name="viewport" content="width=device-width, initial-scale=1.0">//
	
	<link>

	还可以引入css样式或js脚本，这将在后面的内容讲述（这不是个笔记吗
</head>

<body> 是 HTML 文档中用于定义网页主要内容的标签，如文本、图片、表格、链接、视频等
它有一些属性，但大部分已过时，一般用css代替，这里不展开

<h1></h1>~<h6></h6>标题标签，共有六个，默认字体大小递减，不要滥用标题标签
eg:
<h1>文章标题</h1>
<h2>章节一</h2>
<h3>子章节 1.1</h3>
<h3>子章节 1.2</h3>
<h2>章节二</h2>

<p></p>段落标签，表示一段独立的文本内容

<b>加粗文字，视觉强调，无语义
<strong>加粗文字，同时表示重要性（语义化）

<i>斜体文字，视觉强调，无语义
<em>斜体文字，同时表示强调（语义化）

<u>添加下划线，视觉强调，无语义
<s>添加删除线

<br>换行
<hr>分割线

eg：
<p>这是一个段落，这是<b>粗体</b>，这是<i>斜体</i>，这是<strong>重要内容</strong>,这是<em>重要内容</em>,这是<u>下划线</u>这是<s>删除线</s></p>


<a>超链接
使用方法
<a href="目标URL">链接文本</a>//点击“链接文本”，就会跳转到目标URL
a标签还有target属性，用于指定链接的打开方式：
_self（默认值）：在当前窗口打开。
_blank：在新窗口或标签页打开。
_parent：在父框架中打开（用于框架）。
_top：在整个窗口中打开，取消框架。
title属性，提供额外信息，鼠标悬停时会显示


<img>标签，单标签，用于将图片嵌入网页，常见属性：
src：必须属性，指定图片路径，可以是相对路径或绝对路径
alt：图片加载失败时的显示的替代文本
width和height：设置图片的宽度和高度，单位可以是像素（px）或百分比（%）,只修改一个另一个成比例增减
title：鼠标悬停时的文本
eg：
<img src="image/photo.jpg" alt="图片加载失败了喵~" width="500" title="这是图片">


块元素和行内元素
块元素可以包含块级元素和行内元素，行内元素只能包含行内元素
<div>无语义块级标签，一般当做容器
<span>行内元素
